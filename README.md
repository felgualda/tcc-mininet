Experimentos no emulador Mininet, compondo meu trabalho de conclusão de curso.

## Comandos

```sudo mn -c```: limpa interfaces virtuais, tabelas de roteamente e lixo deixado na execução anterior.

```sudo python3 [arquivo.py]```: executa o código em Python com privilégios de administrador.
